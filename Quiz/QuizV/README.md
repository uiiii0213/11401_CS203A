[Quiz V] 樹、堆積與圖論術語 (Tree/Heap/Graph)
這份測驗強化了對資料結構基礎定義的掌握：

核心術語

Tree：Root, Edge, Degree, Height, Leaf Node, Siblings.

Graph：Vertex, Edge with Weight (Cost).

關鍵技術：樹的轉換 (Tree Conversion)

轉換邏輯：將「多叉樹 (General Tree)」轉換為「二元樹 (Binary Tree)」。

方法：採用 Left-Child Right-Sibling (LCRS) 法，只保留第一個孩子與右側兄弟的連結，有效簡化資料結構的複雜度。
